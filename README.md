# Reponsive Portfolio 
A responsive static portfolio website created using HTML and CSS.  

# Features
- responsive layout for mobile, laptop and other devices
- Well-structured sections: Home, About, Education, Skills, Projects, Contact
- Flexible layouts for different screen size
- Clean user-interface

# Tech Stack 
HTML5 – Structure and content
CSS3 – Styling 
Flexbox – Layout management
Media Queries – Responsiveness

# Tools
Git – Version control
GitHub – Code hosting 
VS Code - Code Editor
